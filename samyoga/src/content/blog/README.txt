# Blog Content

This folder contains blog posts in Markdown format.

Each blog post should be:
- A `.md` or `.mdx` file
- Named with a slug (e.g., `yoga-as-wellness.md`)
- Include frontmatter with metadata (title, date, excerpt, tags)

Example frontmatter:
```markdown
---
title: "Yoga as Wellness: Movement and Stillness"
date: "2024-01-15"
excerpt: "Exploring the balance between movement and stillness in yoga practice."
tags: ["wellness", "yoga", "philosophy"]
---
```
