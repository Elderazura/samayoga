# Portrait Videos

Place portrait-oriented videos here (height > width).

These videos will be used for:
- Mobile video content
- Social media videos
- Vertical video displays

Recommended formats: `.mp4`, `.webm`
Recommended dimensions: 1080x1350 or similar (4:5 or 9:16 aspect ratio)
Recommended encoding: H.264 (mp4) or VP9 (webm) for web compatibility
