# Brand Logo

Place Samayoga logo files here.

Include:
- `logo.svg` (preferred - scalable)
- `logo.png` (fallback with transparent background)
- `logo-dark.png` (dark variant if needed)
- `logo-light.png` (light variant if needed)

Recommended dimensions: 512x512px or larger for PNG versions
Recommended format: SVG for best quality at all sizes
